## `> 01` &nbsp; About Me

I'm **Thembinkosi Madiba**, a developer and cybersecurity student based in South Africa 🇿🇦. I'm on a dedicated journey to understand the foundations of cybersecurity — how systems work, how attackers think, and how to build defences that hold.

My approach is **hands-on and curiosity-driven**. I don't just read about security concepts — I experiment in labs, break things in safe environments, analyse real-world attack patterns, and document what I learn. Every lab is a step forward.

Alongside my cybersecurity journey, I use my software development skills to **build security-focused tools** — dashboards, automation scripts, and web-based utilities that support analysis and investigations.

> *"The best way to understand security is to think like both the attacker and the defender."*

## `> 03` &nbsp; 🧪 Cybersecurity Labs

I learn best by doing. Here's a snapshot of the types of practical labs and experiments I'm working through:

**🌐 Networking Labs**
- Packet capture and analysis with Wireshark
- Analysing protocols: TCP handshakes, DNS lookups, HTTP traffic
- Identifying anomalies in captured network traffic

**🐧 Linux / System Labs**
- Linux privilege escalation exercises
- File permissions, user management, system hardening
- Bash scripting for automation and security workflows

**🔒 Web Security Labs**
- OWASP Top 10 vulnerability exploration
- SQL injection, XSS, and CSRF exercises in controlled environments
- Manual and tool-assisted web vulnerability testing

**📧 Phishing & Social Engineering Analysis**
- Analysing real-world phishing email samples
- Header analysis, link inspection, and domain investigation
- Documenting attack patterns and indicators of compromise

**💻 Exploitation Labs**
- Using Metasploit for learning exploitation concepts
- CTF (Capture The Flag) challenges to develop problem-solving skills
- Studying CVEs and understanding vulnerability lifecycles

> ⚠️ *All labs are conducted in isolated, legal environments — VMs, offline labs, and authorised platforms only.*
